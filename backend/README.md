# Backend API

This directory contains the FastAPI application source code.

## Quick Start

1.  **Activate Environment**:
    ```bash
    source venv/bin/activate
    ```

2.  **Run Server**:
    ```bash
    uvicorn app.main:app --reload
    ```

## Key Directories

- `app/`: Main application logic.
- `scripts/db_tools/`: Utilities for database management (seeding, inspection).
- `tests/`: Integration and manual tests.

For full project documentation, architecture details, and setup instructions, please refer to the [Main README](../README.md) in the project root.
